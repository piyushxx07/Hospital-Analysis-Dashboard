---

# 📌 Project Overview

Healthcare organizations generate large volumes of operational and patient data every day.

This project transforms raw Excel datasets into meaningful business insights using **Power BI**, helping hospital management monitor:

- 👨‍⚕️ Patient Analytics
- 🩺 Doctor Performance
- 🏥 Hospital Operations
- 💰 Financial Performance
- 💊 Medicine Inventory
- 📅 Appointments
- ⭐ Patient Satisfaction

# 🎯 Business Objectives

The dashboard enables hospital management to:

- Track patient admissions and discharges
- Monitor doctor availability and earnings
- Analyze hospital resource utilization
- Monitor medicine inventory
- Review financial performance
- Improve operational efficiency
- Support data-driven decision making

# 🧹 Data Preparation

The raw datasets contained multiple tables with redundant columns.

Data transformation was performed using **Power Query** to:

- Remove unnecessary columns
- Clean missing values
- Rename columns
- Standardize formats
- Merge related datasets
- Create analytical tables

The final Power BI model contains optimized tables for reporting.

### Final Data Model

- Appointment
- Medical Stock
- Medicine Patient
- Patient
- Patient Tests
- Supplier
- Bills by Type
- Staff
- Beds Info
